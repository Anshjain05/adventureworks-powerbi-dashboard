# AdventureWorks Power BI Dashboard

This repository contains a complete Power BI dashboard built using the AdventureWorks dataset.

## 📊 Overview

The dashboard includes:
- Sales performance by region and category
- Year-over-year sales trends
- Profit margin analysis
- Top-selling products and customers
- Interactive slicers and drill-down capabilities

## 📁 File

- `AdventureWorks_Report_COMPLETE.pbix`: Power BI file with fully developed visualizations and data model.

## 🛠 Tools Used

- Microsoft Power BI
- AdventureWorks sample dataset

## ✅ How to Use

1. Download the `.pbix` file.
2. Open it in Microsoft Power BI Desktop.
3. Explore the dashboard and visuals.
